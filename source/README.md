# Source Notes

This folder preserves the Python and Jupyter Notebook version of the Treasure Hunt Game. The browser demo in the repository root is the easiest version to run quickly, while these files show the original machine-learning workflow behind the project.

- `TreasureMaze.py` defines the maze environment, movement rules, rewards, and game state.
- `GameExperience.py` stores past experiences for replay during training.
- `TreasureHuntGame.ipynb` documents the deep Q-learning workflow used to train the pirate agent.
