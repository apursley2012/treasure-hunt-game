# Portfolio Content: Treasure Hunt Game

## Card Title

Booty on the Brain: Deep Q-Learning Treasure Hunt Game

## Short Summary

An interactive reinforcement-learning maze where a pirate agent learns to reach treasure by balancing exploration, rewards, penalties, and learned path selection.

## Longer Description

This project began as a Python deep Q-learning implementation and was adapted into a static browser demo for easier viewing. The application preserves the original maze logic, movement rules, reward structure, and agent-training goal while adding a polished front-end interface, manual play controls, training metrics, and animated path replay.

## Technologies

HTML, CSS, JavaScript, Python, Jupyter Notebook, NumPy, Keras, Matplotlib, reinforcement learning, deep Q-learning, experience replay.

## Key Features

- 8×8 maze environment
- Pirate agent and treasure goal
- Manual and trained-agent modes
- Reward and penalty tracking
- In-browser Q-learning trainer
- Animated learned-path replay
- Static GitHub Pages deployment

## Skills Demonstrated

Algorithmic thinking, AI implementation, reinforcement-learning concepts, JavaScript architecture, front-end interaction design, technical documentation, static deployment, and project adaptation.


## Enhanced Demo Note

The browser demo expands the original intelligent-agent maze concept into a full playable race. The visitor controls the player, the agent races toward the X treasure, and every new round generates a fresh 8×8 maze that can be solved without a database or backend. Easy, medium, and hard settings change the agent’s speed and exploration behavior while keeping movement limited to valid open cells.
