Project Overview

Booty on the Brain: A Deep Q-Learning Treasure Hunt Game

This project explores how a reinforcement-learning agent can learn to navigate an 8×8 maze in search of hidden treasure. The agent is represented as a pirate, and its objective is to discover an efficient path from the starting area to the treasure while avoiding walls, repeated moves, invalid actions, and wasted steps.

The original implementation used Python, Jupyter Notebook, NumPy, Keras, Matplotlib, a custom maze environment, and an experience-replay helper. This front-end version recreates the same environment rules, reward structure, movement logic, and training goal as a static browser demo that can be opened directly from GitHub Pages.

Core Concept

The maze is made of open paths and blocked cells. The pirate can move left, up, right, or down. Every action returns a result: progress may lead closer to the treasure, repeated or inefficient movement reduces the reward, and invalid movement is penalized more heavily. Over repeated episodes, the agent learns which choices are more valuable and which routes lead to dead ends.

What the Browser Demo Preserves

  • The same 8×8 maze layout from the Python version.
  • The same four-direction movement system: left, up, right, and down.
  • The same terminal goal: reach the treasure cell in the bottom-right corner.
  • The same reinforcement-learning idea of rewards, penalties, exploration, and learned paths.
  • The same no-database behavior: training runs in memory and resets cleanly between sessions.

What Changed for the Showcase Version

The original notebook workflow was translated into a front-end application with HTML, CSS, and JavaScript. Instead of requiring Python, TensorFlow, Keras, or Jupyter, the demo runs entirely in the browser. I also added manual movement controls, keyboard input, a visual maze board, training metrics, an animated learned-path replay, and a small training log so the learning process is easier to understand at a glance.
